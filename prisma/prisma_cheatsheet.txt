When using insert,update,delete use this type of raw query:
const trunkNo = 'TRK001'
const status = 'active'

await prisma.$executeRaw`
  INSERT INTO records_master (trunk_no, status) 
  VALUES (${trunkNo}, ${status})`

Prisma will automatically bind variable for you, so dont worry about SQL injection


use `queryRaw` when you want to select just query
use `executeRaw` when you want to insert,update,delete just query
